# Accessible study tool

## Features
 - flash cards
 - notes
 - quiz
 - math mode
 - sounds

Account/name system for loading files

## Flash cards
 - group by subject/different sets of cards
 - be able to flip cards
 - save/load system
 - interface for user to enter flash cards

## Notes
 - writing to file
 - save/load system

## Quizzes
 - tests you on flash cards
 - pick a specific set/subject, or random
 - save results to file
 - timer option
 - mixed vs mcq vs frq modes

## Math mode
 - recall mode
 - math notes essentially
